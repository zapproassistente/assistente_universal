🧠 Painel Mestre de Comandos – Assistente Universal

🚦 Fluxo básico Git VPS

Use os scripts no momento certo:
  • 🔼 git_push.sh → Envia mudanças da VPS para o GitHub
  • 🔽 git_pull.sh → Atualiza a VPS com o que está no GitHub
  • 🔁 git_sync.sh → Faz push e pull num só passo (opcional)
./scripts/git_push.sh
./scripts/git_pull.sh
./scripts/git_sync.sh

🛠️ Setup e manutenção

Comandos do dia a dia para gerenciar seu ambiente
sudo apt update && sudo apt upgrade -y
sudo apt install docker-compose nginx git unzip curl zip -y
chmod +x scripts/*.sh

🔥 Rebuild Docker (sem cache)

Quando atualizar o código e quiser reiniciar o container:
docker-compose down && docker-compose up --build -d

📦 Backup do projeto zipado

Salva um .zip com a data atual
zip -r assistente_universal_$(date +%F).zip . -x "*.git*"
